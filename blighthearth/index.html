Blight Hearth

Defend the castle from a massive attack of goblins.

Unreal Engine Blueprints Tower Defense
Trailer / Gameplay Video
Description

You are the last line of defense for the castle, defend from 50 waves of goblins using upgradable weapons and placing towers that are bought from the slime merchant. Beat all 3 maps to defeat those goblins once and for all! Made as the final project for the AIE game programming course.
