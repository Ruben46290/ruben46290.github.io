Title: Peel the Pressure

// Gameplay Information / Backstory
Description 2: After a year long fight with the skeletons ended with a peace treaty, all the mightiest warriors have to settle down on the farm and serve the skeletons. Play Co-op or single player and fulfill all the skeletons orders in all 5 levels to become the ultimate warrior farmer.

Screenshots
peelthepressure/screenshots/example.png

Features
Online Multiplayer - Real-time 2 player co-op via Epic Online Services with a lobby browser
Time Pressure - Race against the clock to fulfill customer orders before time runs out
Customizable Controls - Full key bind remapping with mouse sensitivity options 
Single player Mode - Play solo in a dedicated single player mode where you have more time to fulfill orders

// Technical Information
Developed using *Unreal Engine 5* mainly using C++ with some blueprints. Online play is powered by *Epic Online Services (EOS) for cross-network session discovery and P2P connections. This project features real-time replication and functional server searching & joining.
