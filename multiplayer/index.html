Features

🌐

Online Multiplayer

Real-time 2-player co-op via Epic Online Services with lobby browser and LAN support.

⏱️

Time Pressure

Race against the clock to fulfill customer orders before patience runs out.

🎮

Customizable Controls

Full keybind remapping with mouse sensitivity and invert options.

🧃

Singleplayer Mode

Play solo with a dedicated singleplayer mode and separate progression.

Built With

Developed entirely in Unreal Engine 5 using a mix of C++ and Blueprints. Multiplayer is powered by Epic Online Services (EOS) for cross-network session discovery and P2P connections. The project features custom replication systems, an Enhanced Input keybind system, and a save/load settings pipeline.
