A fast-paced co-op multiplayer game built in Unreal Engine 5 featuring real-time online play via Epic Online Services.

Description

After a year long fight with the skeletons ended with a peace treaty, all the mightiest warriors have to settle down on the farm and serve the skeletons. Play co-op or single player and fulfill all the skeletons' orders in all 5 levels to become the ultimate warrior farmer.

Built With

Developed using Unreal Engine 5 mainly using C++ with some Blueprints. Online play is powered by Epic Online Services (EOS) for cross-network session discovery and P2P connections. This project features real-time replication and functional server searching & joining.
